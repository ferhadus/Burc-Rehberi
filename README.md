```bash
# Clone this project
$ git clone https://github.com/ferhatiltas/Burc-Rehberi
# Access
$ cd Burc-Rehberi

# Pull dependencies
$ flutter pub get

# Run the project (emulator or physical device needs to be connected first)
$ flutter run

# Build an apk
$ flutter build apk --split-per-abi
```
